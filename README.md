# Pulp Magazine Shelf

A static, iPad-friendly comic and pulp magazine viewer for the Internet Archive Pulp Magazine Archive.

## Features

- Bookshelf view powered by Internet Archive advanced search.
- Search and sorting across the collection.
- Full-screen reader for scanned pages.
- Scan enhancement controls for contrast, brightness, saturation, warm tone, invert, fit, and rendered page width.
- Remembers the last page per book in local storage.

## Local Use

Open `index.html` in a browser, or serve the folder with any static server.

```sh
python3 -m http.server 8080
```
